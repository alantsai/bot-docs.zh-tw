使用 [Bot Framework 模擬器](~/bot-service-debug-emulator.md)驗證 Bot 的部署。 

在模擬器的網址列中輸入 Bot 的**傳訊端點**。 如果您是使用 Bot Framework SDK 建置 Bot，端點應該會以 */api/messages* 結尾。

範例： `https://<appname>.azurewebsites.net/api/messages`
